# Infinity Facilities Management Website

## Project Overview
This is the official website for Infinity Facilities Management, a premier property management venture in Raipur, Chhattisgarh. The website showcases services, projects, and company information.

## Recent Updates (December 2025)

### Global Navigation & Styling
- **Animated Hamburger Menu**: Implemented a consistent animated hamburger menu (transforming to 'X') across all pages (`index.html`, `about.html`, `services.html`, `projects.html`, `contact.html`).
- **Navbar Consistency**: Standardized the navigation bar styling, including glassmorphism effects and active link highlighting.
- **Color Scheme**: Unified the global color scheme to match the `contact.html` palette (Primary: `#216fb4`, Accent: `#d4af37`).

### Page-Specific Fixes
- **Projects Page (`projects.html`)**:
    - Updated the project data structure to support distinct `logo` and `image` fields.
    - Updated project images to use high-quality assets where available.
    - Fixed the navigation bar to match the global standard.
- **About Us Page (`about.html`)**:
    - Restored `data-aos` animations for the "Meet The Team" section.
- **Services Page (`services.html`)**:
    - Applied bold formatting to key service list items for better readability.

## Technology Stack
- **Frontend**: HTML5, Tailwind CSS (CDN)
- **Animations**: AOS (Animate On Scroll) Library
- **Fonts**: Google Fonts (Inter, Playfair Display)

## Setup & Running
To run the project locally:
1. Open the project folder in a terminal.
2. Run a local server (e.g., `python -m http.server 8000`).
3. Navigate to `http://localhost:8000` in your browser.
