# Infinity Facilities Management - Technical Manual

## Project Overview
This is the official website for Infinity Facilities Management, a static web application showcasing services, projects, and company information.

## Technology Stack
*   **HTML5**: Semantic structure of the pages.
*   **CSS3**: Custom styling for layout and design.
*   **JavaScript**: Interactive elements (if any).

## Project Structure
*   `index.html`: Home page.
*   `about.html`: Company information and "Meet The Team" section.
*   `services.html`: Details of services offered.
*   `projects.html`: Showcase of completed projects.
*   `contact.html`: Contact form and information.
*   `assets/`: Directory containing images, styles, and scripts.

## Deployment
The website is static and can be deployed to any static site hosting service (e.g., GitHub Pages, Netlify, Vercel) or served via a standard web server (Apache, Nginx).
